# Applitools

Applitools Ruby SDK.

## Installation

Add this line to your application's Gemfile:

    gem 'eyes_selenium'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install eyes_selenium

## Usage
Please check the applitools website for usage instructions: http://www.applitools.com
